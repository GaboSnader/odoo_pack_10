Creacion De Productos Tipo Paquetes

Metodo de Creacion de Paquetes

Declaracion de Paquetes

Se utiliza para especificar que el producto creado es un paquete en el cual se podran agregar mas productos.

Lista de Productos

Al activar la opcion de paquete se mostrara una nueva pesta√±a de "Productos", en donde se agregan los productos que desea tener dentro del paquete. Nota: No se pueden introducir paquetes.

Precio Basado en Componentes

Si desea que el precio fijo de paquete sea la suma de los componentes del mismo se activa la opcion "Precio Basado en Componentes", de lo contrario de tendra que poner el costo en "Precio de Venta".

Paquetes en Pedidos de Venta

Agregar paquetes dentro de un pedido de venta

Al agregar algun paquete dentro de un pedido de venta se puede mostrar de dos maneras diferentes dependiendo de la declaracion del precio del paquete:

Paquete Basado en Componentes

Si el paquete esta declarado con precio basado en componentes, al guardar el pedido de venta, el subtotal del paquete se mostrara con un valor igual a cero, miestras que apareceran los componentes del paquete y cada uno tendra como subtotal el valor asignado dependiendo la cantidad pedida y el precio del componente.

Paquete Basado en Precio de Venta

Cuando el paquete no esta declarado con precio basado en componentes, el paquete aparecera con el precio de venta asignado por el usuario, cuando se guarde el pedido apareceran los componentes del producto pero, de lo contario a lo anterior, estos tendran como subtotal un valor igual a cero cada uno.
